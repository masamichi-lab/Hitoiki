# プライバシーポリシー

最終更新日: 2026年5月5日

## 収集する情報

ひといき（iOS版・Android版を含む、以下「本アプリ」）は、利用者から個人情報を収集しません。

本アプリで記録された内容（怒りの記録、子どもの名前、きっかけ、メモ、気持ち、行動など）は、すべて利用者のデバイス上に保存されます。開発者はこれらのデータを閲覧できません。

### iOS版

データはデバイス上の SwiftData に保存され、利用者ご自身の Apple ID に紐づく **iCloud プライベートデータベース** にのみ同期されます。同じ Apple ID でサインインした他の iPhone / iPad とのみ同期されます。

### Android版

データはデバイス上の Room（端末内 SQLite データベース）にのみ保存されます。**クラウド同期は行いません。** 他の端末との同期もバックアップも行わず、データは利用者の端末を離れません。

## 第三者への提供

本アプリは、利用者のデータを第三者に提供しません。

## 解析・トラッキング

本アプリは、解析ツールやトラッキング SDK を一切使用していません。広告も配信しません。

## 通信

- **iOS版** が行う通信は、Apple の iCloud / CloudKit との同期通信のみです。それ以外の外部サーバーへの通信は行いません。
- **Android版** はインターネット通信を一切行いません。

## データの削除

### iOS版

iCloud に保存されたデータは、デバイスからアプリを削除しても iCloud 上には残ります。完全に削除したい場合は、iOS の「設定 → Apple ID → iCloud → iCloud を使用している App → ひといき → 削除」を行ってください。

### Android版

データはデバイス内にのみ保存されています。アプリをアンインストールするか、Android の「設定 → アプリ → ひといき → ストレージとキャッシュ → ストレージを消去」でデータが完全に削除されます。
