# プライバシーポリシー

最終更新日: 2026年4月30日

## 収集する情報

ひといき（以下「本アプリ」）は、利用者から個人情報を収集しません。

本アプリで記録された内容（怒りの記録、子どもの名前、きっかけ、メモ、気持ち、行動など）は、利用者のデバイス上の SwiftData に保存され、利用者ご自身の Apple ID に紐づく **iCloud プライベートデータベース** にのみ同期されます。開発者はこれらのデータを閲覧できません。

## 第三者への提供

本アプリは、利用者のデータを第三者に提供しません。

## 解析・トラッキング

本アプリは、解析ツールやトラッキング SDK を一切使用していません。広告も配信しません。

## 通信

本アプリが行う通信は、Apple の iCloud / CloudKit との同期通信のみです。それ以外の外部サーバーへの通信は行いません。

## データの削除

iCloud に保存されたデータは、デバイスからアプリを削除しても iCloud 上には残ります。完全に削除したい場合は、iOS の「設定 → Apple ID → iCloud → iCloud を使用しているApp → ひといき → 削除」を行ってください。
