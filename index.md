# ひといき (Hitoiki)

子育ての怒りや戸惑いを書きとめて、自分にひといきの時間を渡すための iOS / Android アプリです。

- 怒った瞬間を 3 タップで記録
- 深呼吸 → 気持ちの確認 → 次の一歩、3 ステップのひといきガイド
- 今日 / 今週 / 今月 / 今年の傾向をふりかえり
- iOS版は iCloud のプライベートデータベースで端末間同期。Android版は端末内のみに保存（クラウド同期なし）。いずれも開発者はあなたのデータにアクセスできません
