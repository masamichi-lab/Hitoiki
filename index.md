# ひといき (Hitoiki)

子育ての怒りや戸惑いを書きとめて、自分にひといきの時間を渡すための iOS アプリです。

- 怒った瞬間を 3 タップで記録
- 深呼吸 → 気持ちの確認 → 次の一歩、3 ステップのひといきガイド
- 今日 / 今週 / 今月 / 今年の傾向をふりかえり
- 全データは iCloud のプライベートデータベースで同期。開発者はあなたのデータにアクセスできません
