# サポート

ひといきに関するご質問・不具合報告・ご要望は、GitHub Issues にてお願いします。

- <https://github.com/masamichi-lab/Hitoiki/issues/new>

GitHub アカウントが必要です。アカウントをお持ちでない方は無料で作成できます。

## よくある質問

### データはどこに保存されますか？

- **iOS版** — 利用者ご自身の iCloud プライベートデータベースにのみ保存されます。
- **Android版** — 利用者の端末内（Android のアプリ内ストレージ）にのみ保存されます。クラウド同期は行いません。

いずれも開発者はあなたのデータを見ることができません。

### 別の端末でも使えますか？

- **iOS版** — 同じ Apple ID でサインインしていれば、iPhone / iPad 間で自動的に同期されます。
- **Android版** — 端末間の同期は行いません。各端末で個別にご利用ください。

### iCloud をオフにしている場合は？（iOS版）

端末ローカルにのみ保存され、他のデバイスとは同期されません。

### iOS と Android の間でデータを同期できますか？

現在、プラットフォーム間のデータ同期には対応していません。
